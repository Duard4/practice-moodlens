# MoodLens: Платформа для аналізу та обміну кінорецензіями

MoodLens - це платформа для створення, збереження та аналізу кінорецензій. Користувачі можуть переглядати архіви, ставити реакції, коментувати та отримувати аналітику настроїв.

## Встановлення

1. Клонувати репозиторій:

    ```sh
    git clone <repository-url>
    cd practice-moodlens
    ```

2. Встановити залежності:

    ```sh
    npm install
    ```

## Запуск проекту

1. Запустити проект у режимі розробки:

    ```sh
    npm run dev
    ```

2. Відкрити [http://localhost:5173](http://localhost:5173) у браузері.

## Скрипти

- `npm run dev`: Запускає проект у режимі розробки.
- `npm run build`: Збирає проект для продакшну.
- `npm run lint`: Запускає ESLint для перевірки коду.
- `npm run preview`: Запускає зібраний проект для перегляду.

## Структура проекту

- src: Вихідний код проекту.
  - `components/`: Компоненти React.
  - `data/`: JSON файли з даними.
  - `hooks/`: Кастомні хуки.
  - `pages/`: Сторінки додатку.
  - `redux/`: Redux store та slice.
  - `styles/`: CSS модулі.
- public: Публічні файли.
- index.html: Головний HTML файл.
- package.json: Файл конфігурації npm.
- tailwind.config.js: Конфігурація Tailwind CSS.
- vite.config.js: Конфігурація Vite.

## Використані технології
